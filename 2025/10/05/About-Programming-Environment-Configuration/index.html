Judge (power shell script, in Windows system)

This a simple judge, which can examine the difference between $.out$ and $.ans$ file, then output the status into $.status$ file.

Input

argument type description
0 PATH The abosolute path of .exe file
1 PATH The abosolute path of .in file
2 PATH The abosolute path of .out file
3 PATH The abosolute path of .ans file
4 PATH The abosolute path of .status file
5 number The time limit
6 PATH The abosolute path of error file, which can store the error message during the process. This file should will be created in the same root directory as judge automatically.
