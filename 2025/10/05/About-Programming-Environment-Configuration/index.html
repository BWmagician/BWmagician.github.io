Judge (power shell script, in Windows system)

This a simple judge, which can examine the difference between $.out$ and $.ans$ file, then output the status into $.status$ file.

Input

argument type description
0 PATH The abosolute path of .exe file
1 PATH The abosolute path of .in file
2 PATH The abosolute path of .out file
3 PATH The abosolute path of .ans file
4 PATH The abosolute path of .status file
5 number The time limit
6 PATH The abosolute path of error file, which can store the error message during the process. This file should will be created in the same root directory as judge automatically.

Example Presenter (vim configuration,_vimrc)

vim use $s:$ to present the variable/function that can be called in script only. $g:$ to present the variable/function that can be called both in script and vim. However, if you want to change certain varible permanently, you still need to modify the configuration in the vim script. name|type| usage |description :-:|:--:|:---:|:--: g:datLimit|number|-|maximum number examples in the Example Presenter g:exWidth|number|-|width of example column when calling function g:ModifyWindows() s:RootFolder|string|-|The Example Presenter will automatically create a folder in the same root directory of .cpp file.
