еҶҷеңЁеүҚйқў

зӣ®еүҚеҸӘеҶҷиҝҮеҲҶжІ»пјҢд»ҘеҗҺдјҡиЎҘе……зҡ„ (2025/7/18

и®ҫе®ҡ

иҖғиҷ‘дёҖдёӘжғ…жҷҜпјҢ$\displaystyle f_i=\min_{j<=i}\{f_i+c(j,i)\}$
еӣӣиҫ№еҪўдёҚзӯүејҸеҚіпјҢеҜ№дәҺ$a \leq b, c \leq d$пјҢж»Ўи¶і$c(a,c)+c(b,d) \leq c(a,d)+c(b,c)$пјҢеҲҷ$f$зҡ„еҶізӯ–зӮ№еҚ•и°ғгҖӮеӣӣиҫ№еҪўдёҚзӯүејҸеҸҜд»Ҙи®°дҪңдәӨеҸүдјҳдәҺеҢ…еҗ«

еҶізӯ–зӮ№

еҜ№дәҺ$f_i$еҶізӯ–зӮ№е°ұжҳҜжңҖе°ҸеҢ–$f_i$зҡ„$j$гҖӮ
ж— йңҖйЎҫиҷ‘еҜ№дәҺеҗҢдёҖдёӘ$f_i$жңүеӨҡдёӘ$j$зҡ„жғ…еҶөпјҢдҪ еҸҜд»Ҙд»»йҖүдёҖдёӘ$j$иҝӣиЎҢеҚ•и°ғжҖ§зҡ„йҖ’жҺЁ

еҚ•и°ғжҖ§

еҸҜд»ҘеҸҚиҜҒ
иҖғиҷ‘дёҖдёӘж»Ўи¶іеӣӣиҫ№еҪўдёҚзӯүејҸпјҢеҚҙеҶізӯ–зӮ№дёҚеҚ•и°ғзҡ„жғ…еҶө:
еҒҮи®ҫжңүдёӨдёӘжңҖдјҳзҡ„зҠ¶жҖҒ$f_i,f_p$пјҢд»–д»¬зҡ„еҶізӯ–зӮ№еҲҶеҲ«жҳҜ$j,q$пјҢжҚўиЁҖд№ӢпјҢ$f_i=f_j+c(j,i),f_p=f_q+c(q,p)$пјҢеҚҙдёҚж»Ўи¶іеҚ•и°ғжҖ§пјҢд№ҹе°ұжҳҜиҜҙ$q \leq j \leq i \leq p$
жҲ‘д»¬еҸҜд»ҘжҠҠдёӨиҫ№еҠ иө·жқҘпјҢжңү$f_i+f_p=f_j+f_q+c(j,i)+c(q,p)$
з„¶иҖҢж №жҚ®еӣӣиҫ№еҪўдёҚзӯүејҸжңү$f_j+f_q+c(j,p)+c(q,i) \leq f_j+f_q+c(j,i)+c(q,p)$
жңү$(f_j+c(j,p))+(f_q+c(q,i)) \leq f_j+f_q+c(j,i)+c(q,p)$
жҚўиЁҖд№Ӣ$f_{p\prime}+f_{i\prime} \leq f_p+f_i$
дёҚиҖғиҷ‘зӯүеҸ·еҲҷиҮіе°‘$f_{p\prime}$е’Ң$f_{i\prime}$дёӯзҡ„дёҖдёӘиҫғеҺҹжқҘжӣҙдјҳпјҢеҲҷжӯӨжғ…еҶөдёҚжҲҗз«ӢгҖӮ
иҖғиҷ‘зӯүеҸ·еҲҷжҳҜжҲ‘д»¬дёҠйқўи®Ёи®әзҡ„еҸҜиғҪжңүеӨҡдёӘеҶізӯ–зӮ№зҡ„жғ…еҶөпјҢд»Қз¬ҰеҗҲеӣӣиҫ№еҪўдёҚзӯүејҸгҖӮ

еҲҶжІ»

жҲ‘д»¬зӣҙжҺҘиҖғиҷ‘йҖҡиҝҮеҶізӯ–зӮ№зҡ„еҚ•и°ғжҖ§жқҘдјҳеҢ–dpжҳҜеҫҲеӣ°йҡҫзҡ„пјҢеӣ дёәеҶізӯ–зӮ№еҚ•и°ғдёҚж„Ҹе‘ізқҖе…·дҪ“зҠ¶жҖҒеҜ№еә”зқҖдёҖдёӘе…·жңүеҮ№еҮёжҖ§иҙЁзҡ„еҮҪж•°гҖӮжҚўиЁҖд№ӢпјҢеҰӮжһңдҪ жғізЎ®е®ҡжҺЁеҜјеҮәжҹҗдёӘзҠ¶жҖҒзҡ„еҶізӯ–зӮ№пјҢдҪ йңҖиҰҒ$O(n)$зҡ„ж—¶й—ҙеҺ»жҺЁз®—гҖӮ

е…·дҪ“жҖқи·Ҝ

жҲ‘д»¬иҰҒжіЁж„ҸеҲ°еҚ•и°ғжҖ§иЎЁжҳҺдәҶдёӨдёӘдҝЎжҒҜпјҢжҹҗдёӘзҠ¶жҖҒзҡ„еҶізӯ–зӮ№дёҖж—Ұз®—еҮәпјҢеҲҷе…¶еүҚйқўзҡ„зҠ¶жҖҒзҡ„еҶізӯ–зӮ№еңЁе…¶еҶізӯ–зӮ№д№ӢеүҚпјҢеҸҚд№ӢпјҢе…¶еҗҺйқўзҡ„зҠ¶жҖҒеңЁе…¶еҶізӯ–зӮ№д№ӢеҗҺгҖӮиҝҷдёҚе°ұе®ҢзҫҺз¬ҰеҗҲдәҶеҲҶжІ»зҡ„зү№зӮ№гҖӮ

е…·дҪ“жқҘиҜҙпјҢ жҜҸж¬Ўз»ҷе®ҡйңҖиҰҒжұӮи§Јзҡ„зҠ¶жҖҒиҢғеӣҙ$(l,r) $пјҢе’ҢжҺЁеҜјиҝҷдәӣзҠ¶жҖҒзҡ„еҶізӯ–зӮ№зҡ„иҢғеӣҙ$(L,R)$гҖӮи®©$mid=\frac{l+r}{2}$пјҢжҲ‘д»¬жҜҸж¬Ўе…ҲжҡҙеҠӣжұӮи§Ј$mid$зҠ¶жҖҒзҡ„еҶізӯ–зӮ№зҡ„дҪҚзҪ®$p$гҖӮеҲҷ$(l,mid-1)$еҜ№еә”зҡ„еҶізӯ–зӮ№иҢғеӣҙжҳҜ$(L,p)$пјҢ$(mid+1,r)$еҜ№еә”зҡ„еҶізӯ–зӮ№иҢғеӣҙжҳҜ$(p,R)$

еӨҚжқӮеәҰ

йҰ–е…ҲиҝҷжҳҜдёӘеҲҶжІ»пјҢжүҖд»ҘеӨҚжқӮеәҰе·Із»ҸжқҘеҲ°дәҶ$O(n \log n)$пјҢеҰӮжһңеҶізӯ–зӮ№зҡ„иҢғеӣҙдёҚзӣёдәӨпјҢеҲҷеӨҚжқӮеәҰжҳҜ$O(n \log n)$гҖӮжҲ‘д»¬иҝҷйҮҢеҒҮи®ҫ$n$жҳҜзҠ¶жҖҒзҡ„жҖ»ж•°гҖӮ

дҪҶжҳҜеҶізӯ–зӮ№зҡ„иҢғеӣҙе®һйҷ…дёҠжңүдёҖдёӘзӮ№зҡ„зӣёдәӨгҖӮеҸҲз”ұдәҺиҝҷжҳҜдёӘеҲҶжІ»пјҢжүҖд»ҘеҢәеҹҹеҲ’еҲҶдёҚдјҡи¶…иҝҮ$2n$пјҢзӣёдәӨзҡ„иҢғеӣҙжҖ»е’ҢжҳҜдёҚдјҡи¶…иҝҮ$2n$зҡ„гҖӮ

дәҺжҳҜеҒҡдёҖж¬ЎеҲҶжІ»зҡ„жҖ»еӨҚжқӮеәҰиҝҳжҳҜ$O(n \log n)$

йқһеёёеҘҪзҡ„дҫӢйўҳ

P5574 [CmdOI2019] д»»еҠЎеҲҶй…Қй—®йўҳ

ж»Ўи¶іеӣӣиҫ№еҪўдёҚзӯүејҸ

$c(x,y)$д»ЈиЎЁзҡ„жҳҜ$(x,y)$иҢғеӣҙеҶ…зҡ„йҖҶеәҸеҜ№дёӘж•°
и®Ўз®—еҰӮдёӢзҡ„жғ…еҶөзҡ„йҖҶеәҸеҜ№пјҢиҝҷйҮҢз”ЁеӨ§еӯ—жҜҚд»ЈиЎЁиҢғеӣҙеҶ…зҡ„йҖҶеәҸеҜ№ж•°пјҢз”Ёз®ӯеӨҙиЎЁзӨәдёҖиҫ№еңЁе·Ұдҫ§еҢәеҹҹдёҖиҫ№еңЁеҸіиҫ№еҢәеҹҹзҡ„йҖҶеәҸеҜ№

дәӨеҸү

$c(a,c)=A+B+A \to B$
$c(b,d)=B+C+B \to C$

еҢ…еҗ«

$c(a,d)=A+B+C+A \to B+A\to C+B \to C$
$c(b,c)=B$

еҲҷ$c(a,c)+c(b,d)=A+2B+C+A \to B+B \to C\leq A+2B+C+A \to B+A \to C+B\to C=c(a,d)+c(b,c)$пјҢиҜҙжҳҺдәӨеҸүдјҳдәҺеҢ…еҗ«

з»ҹи®ЎйҖҶеәҸеҜ№ж•°

з”ұдәҺжҲ‘д»¬з”Ёзҡ„жҳҜеҲҶжІ»пјҢжҢүз…§еҲҶжІ»йЎәеәҸеҲҶеүІзҡ„иҢғеӣҙе®һйҷ…дёҠжҳҜеҫҲи§„еҫӢзҡ„гҖӮ
иҝҷйҮҢеҸӮиҖғдәҶйўҳи§Ј P5574 гҖҗ[CmdOI2019]д»»еҠЎеҲҶй…Қй—®йўҳгҖ‘зҡ„еҶҷжі•
жҲ‘д»¬еҸҜд»Ҙз”Ёзұ»дјјдәҺиҺ«йҳҹзҡ„еҸҢжҢҮй’Ҳз§»еҲ°зӣ®ж ҮиҢғеӣҙжқҘи®Ўз®—йҖҶеәҸеҜ№гҖӮ

еӨҚжқӮеәҰ

е…·дҪ“еҲҶжІ»зҡ„жҹҗдёҖж®өиҢғеӣҙжңҖеӨҡеҸӘдјҡз»ҹи®Ўеёёж•°йҒҚгҖӮе…·дҪ“дёәжҜҸж®өиҢғеӣҙжҢҮй’ҲдјҡжҡҙеҠӣиө°е·ҰеҚҠиҫ№пјҢиө°еҗ‘еҲҶеүІзҡ„е·ҰеҢәй—ҙжҲ–д»Һе·ҰеҢәй—ҙеӣһжқҘж—¶иҰҒејәеҲ¶з§»еҠЁеӣӣеҲҶд№ӢдёҖзҡ„иҢғеӣҙпјҢиө°еҗ‘еҸіеҢәй—ҙеҗҢзҗҶгҖӮйӮЈд№ҲжҢҮй’Ҳз§»еҠЁзҡ„ж¬Ўж•°ж—¶$O(n \log n)$зә§еҲ«зҡ„

з”ұдәҺжҲ‘д»¬з»ҹи®ЎйҖҶеәҸеҜ№иҰҒз”ЁеҲ°ж ‘зҠ¶ж•°з»„пјҢеҲҷжҢҮй’Ҳз§»еҠЁеёҰжқҘзҡ„еӨҚжқӮеәҰжҳҜ$O(n \log^2 n)$

жҖ»еӨҚжқӮеәҰ

д»–иҜҙз»ҷеҢәй—ҙеҲҮ$k$еҲҖпјҢе®һйҷ…дёҠеҸӘиҰҒеҒҡ$k$ж¬ЎеҲҶжІ»еҚіеҸҜпјҢе…·дҪ“зңӢд»Јз ҒгҖӮ
дәҺжҳҜжҖ»еӨҚжқӮеәҰжҳҜ$O(kn \log^2 n)$
